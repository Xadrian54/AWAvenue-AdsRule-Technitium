## Visual Studio (2026)
### Note: Domains listed without wildcards (*.), as Technitium matches subdomains automatically.

1. Open file
2. Press `Ctrl + H`
3. Enable **Regular Expression** mode (`.*` icon or `Alt + E`)
4. Use the following values:

```text
Search: ^\.
Replace with nothing 
```
5. Click Replace All
